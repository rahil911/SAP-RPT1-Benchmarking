## File Formats

**Mermaid (.mmd)**: Source diagram files (editable)
**SVG (.svg)**: Rendered vector graphics (high-resolution, scalable)

All visuals follow the design system:
- **Colors**: Navy Blue (#1E3A8A), Slate Gray (#64748B), Teal (#14B8A6)
- **Typography**: Helvetica Neue, Arial, sans-serif
- **White Space**: 40-50% minimum
- **Style**: Professional, academic, publication-quality
