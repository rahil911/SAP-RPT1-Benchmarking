## Primary Reference

**ConTextTab: Table Foundation Model for Relational Tables**
- **arXiv ID**: 2506.10707v4
- **Authors**: SAP AI Research Team
- **Publication Date**: November 2025
- **File**: [../2506.10707v4.pdf](../2506.10707v4.pdf)

**Paper Summary**:
- Introduces SAP RPT-1 (ConTextTab), first enterprise relational foundation model
- Zero-shot prediction capabilities on tabular data
- Semantic understanding of table structures
- Benchmark results on TabArena and internal datasets
